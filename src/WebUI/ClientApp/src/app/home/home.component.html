Web app creata con Angular, ASP.NET Core, Entity Framework Core e Database SQL.

Features disponibili:

  • Registrazione e autenticazione.
  • Possibilità di implementazione di ruoli per riservare le azioni più importanti solo a specifici utenti.
  • Consultazione e gestione di un database contenente prodotti in molteplici formati divisi per gruppi e categorie.
  • Supporto di molteplici lingue per i dati salvati nel database.
  • Possibilità di ricerca dei dati tramite parole-chiave.
  • API REST pubblica.
